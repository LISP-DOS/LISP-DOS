Módulo VGA e Vídeo

Documentação da Camada de Abstração de Hardware (HAL) - LISP-DOS

Visão Geral da Arquitetura de Vídeo

Framebuffer Limine (Linear Framebuffer): Diferente de sistemas DOS clássicos que utilizam MMIO na porta `0xB8000` (Hardware VGA de Modo Texto Nativo), o LISP-DOS opera em Long Mode via UEFI/BIOS usando o protocolo Limine. Por isso, a saída de vídeo é manipulada desenhando pixels brutos (Rasterização) numa matriz contígua chamada de Framebuffer.

A emulação de "Terminal Texto" do nosso ambiente virtual funciona extraindo bitmaps de uma matriz de fontes de 8x8 pixels (font8x8_basic.h), escalando a arte via software (com o multiplicador FONT_SCALE = 2) e despejando na memória de vídeo requisitada pela tag limine_framebuffer_request.

Variáveis Globais e Estruturas

Protocolo Limine

volatile struct limine_framebuffer_request framebuffer_request

Força o Bootloader a iniciar o hardware gráfico e reportar os endereços físicos. Em seguida, as variáveis globais de estado cuidam de controlar o cursor virtual em terminal_row e terminal_column.

O sistema também faz o log duplo. Cada letra processada pelo vídeo também é encaminhada simultaneamente para a saída Serial Port (0x3F8) para propósitos de depuração headless.

Funções da API (vga.h)

void vga_init(void)

Extrai o ponteiro fb (Framebuffer Base Address) da resposta do Limine e chama vga_clear() pela primeira vez.

void vga_clear(void)

Itera sobre todos os fb->width * fb->height pixels repintando a tela com a cor predefinida em bg_color. Reposiciona o cursor em (0,0).

void vga_putchar(char c)

O coração do módulo. Decodifica o caractere (c) da tabela ASCII e varre os bits em font8x8_basic[(uint8_t)c]. Também processa lógicas de retrocesso \b (Backspace apagando pixels anteriores), e \n (Newline e Wrap-around limitante).

void vga_set_color(uint8_t fg, uint8_t bg)

Interpreta um código padrão VGA (0 a 15) e converte na hora para inteiros RGB puros (0xRRGGBB). Por exemplo, código VGA 2 converte para 0x00AA00 (Verde) internamente no buffer da placa de vídeo.

void vga_set_cursor(size_t row, size_t col)

Permite a programas utilitários forçarem as coordenadas do Cursor de forma programática. (Nota: o cursor atual na interface é renderizado pelo Ring 3 desenhando espaços de fundo invertido por cima dessa matriz subjacente).

size_t vga_get_max_rows(void) / size_t vga_get_max_cols(void)

Retorna as dimensões matemáticas disponíveis baseada na resolução atual provida pelo Limine e na escala aplicada à fonte (Width / (8 * FONT_SCALE)).
