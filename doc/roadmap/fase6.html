Fase 6: Portabilidade ARM e Raspberry Pi 4

A Fase 6 marca o momento em que o LISP-DOS se torna um sistema multiplataforma real. O objetivo é realizar o porte seguro do Kernel Core (atualmente preso à arquitetura Intel x86_64) para a arquitetura ARMv8-A (AArch64), com foco no paradigma Bare-Metal do Raspberry Pi 4 (BCM2711).

Diferente de PCs tradicionais, o Raspberry Pi usa um modelo focado na GPU (VideoCore) para o processo de boot e possui mapas de memória engessados fisicamente. Esta fase exige a construção de uma nova Hardware Abstraction Layer (HAL) mantendo o motor do LISP intacto.
